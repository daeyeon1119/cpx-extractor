CPX 추출기 — 오프라인 설치형 앱 (PWA)
=======================================

이 폴더 하나가 "설치되는 앱"입니다. 인터넷 없이도 완전히 동작하며,
한 번 설치하면 홈 화면/바탕화면에 아이콘이 생기고 별도 창으로 열립니다.

폴더 구성
---------
  index.html              앱 본체
  manifest.webmanifest    앱 정보(이름/아이콘/색)
  sw.js                   오프라인 캐시(서비스 워커)
  icons/                  앱 아이콘들
  serve.bat               내 PC에서 바로 설치하기용 실행 파일

★ 중요 (꼭 읽어주세요)
----------------------
  index.html 을 그냥 더블클릭해서 열면(주소가 file:// 로 시작) 앱은
  "실행"되지만 "설치"는 되지 않습니다. 브라우저 보안 규칙 때문에
  설치 기능은 반드시 웹 주소(http/https 또는 localhost)로 열렸을 때만
  작동합니다. 그래서 아래 방법 중 하나로 열어주세요.


■ 방법 A — 내 윈도우 PC에 설치 (가장 간단, 계정 불필요)
-------------------------------------------------------
  1) serve.bat 을 더블클릭합니다.
     (검은 창이 뜨고 크롬/엣지가 http://localhost:8765 로 자동으로 열립니다.)
  2) 화면 오른쪽 위 [설치] 버튼을 누릅니다.
     버튼이 안 보이면 주소창 오른쪽의 설치(⊕) 아이콘,
     또는 브라우저 메뉴(⋮) → "앱 설치 / CPX 추출기 설치"를 누르세요.
  3) 설치 완료! 이제 검은 창(serve.bat)을 닫아도 됩니다.
     바탕화면/시작 메뉴의 CPX 추출기 아이콘으로 언제든 오프라인 실행됩니다.

  ※ python 이 설치돼 있어야 serve.bat 이 동작합니다.
    (없다면 https://python.org 에서 설치하거나, 방법 B를 사용하세요.)


■ 방법 B — 휴대폰에 설치 / 친구에게 공유 (온라인 주소로 배포)
------------------------------------------------------------
  이 폴더의 파일들을 무료 정적 호스팅에 올리면 주소(URL) 하나가 생기고,
  그 주소를 여는 사람 누구나 설치할 수 있습니다.

  쉬운 호스팅 예:
   - GitHub Pages : github.com 에 폴더 내용을 올리고 Pages 기능 켜기
   - Netlify Drop : app.netlify.com/drop 에 이 폴더를 통째로 끌어다 놓기
   - Cloudflare Pages / Vercel 등도 가능

  올린 뒤 생긴 https:// 주소를:
   - 휴대폰에서 열고 →
       * 아이폰(사파리): 공유 버튼 → "홈 화면에 추가"
       * 안드로이드(크롬): 메뉴(⋮) → "앱 설치 / 홈 화면에 추가"
   - PC에서 열고 → [설치] 버튼 또는 주소창 설치 아이콘

  이 주소를 친구에게 링크로 보내면, 친구도 똑같이 설치해서 씁니다.


저장되는 데이터에 대해
----------------------
  뽑기 진행 상황과 자가 피드백 메모는 "그 기기의 그 브라우저"에만
  저장됩니다. 다른 사람/다른 기기로는 기록이 따라가지 않으며,
  각자 빈 상태로 시작합니다. (앱을 공유하는 것이지 기록을 공유하는 게 아닙니다.)


앱 내용을 수정했다면
--------------------
  index.html 을 고친 뒤에는 sw.js 안의 'cpx-extractor-v1' 을
  'cpx-extractor-v2' 처럼 숫자만 바꿔주세요. 그래야 설치된 앱이
  다음 실행 때 새 버전으로 갱신됩니다.
